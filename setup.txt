To setup this project for your own environment:

Ensure you have Node.js installed: https://nodejs.org/en/download/

Make sure you have the Expo Go app installed on your phone as well. For best results on iOS, go to Settings -> Display & Brightness -> Text Size and set the minimum text size.

Once you download the project, open a new terminal window in the project directory and run npm install. This will download all the dependencies for the project.

Now run the command npx expo start. Scan the QR code that is presented and it will open the app in the Expo Go app.
